# Dungeon of Cor

Explore a procedural dungeon, kill all monsters to proceed to the next level. Go as deep as you can!

Controls:
```
[W A S D] - Movement
[Left Mouse Button] - Attack
[C] - Open Map
[ESC] - Open Menu
```
A game made in Godot 2.1 (alpha)
